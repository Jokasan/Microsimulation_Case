
CDRC 2011 OAC Geodata Pack - Leeds (E08000035)

+ Abstract:

The 2011 Area Classification for Output Areas (2011 OAC) is a UK geodemographic classification. This CDRC 2011 OAC Geodata Pack provides a mappable version of this classification for the Local Authority District: Leeds (E08000035)

+ Contents:
	- E08000035_Leeds (.shp,.prj,.dbf,.shx): Shapefile
	- metadata.xml: Meta Data
	- penportraits.pdf: ONS Pen Portraits and Names
	- readme.txt: Information about the CDRC 2011 OAC Geodata
	- Cluster_Names.csv: Descriptions of the classified categories.

+ Attributes in the shapefile:
	- oa_code: Output Area code
	- supgrpcode: Supergroup code for this Output Area
	- supgrpname: Supergroup description for this Output Area
	- grpcode: Group code for this Output Area
	- grpname: Group description for this Output Area
	- subgrpcode: Subgroup code for this Output Area
	- subgrpname: Subgroup description for this Output Area
